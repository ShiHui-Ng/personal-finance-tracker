# Personal Finance Tracker

## <img src="https://img.freepik.com/free-vector/statistics-data-analysis-business-illustration_24908-59546.jpg" width="50"> Description
A Python-based finance tracking application that allows users to:
- Add transactions
- Track income and expenses
- Analyze monthly cashflow
- Visualize financial data using matplotlib

## <img src="https://img.freepik.com/free-vector/earth-globe-with-light-bulb-icon_1308-130718.jpg" width="50"> Ideal for learning basic Python concepts such as file handling, functions, and data structures. 

## Features
* Monthly cashflow analysis
* Category-based spending breakdown
* Data visualization dashboard
* CSV-based data storage

## <img src="https://img.freepik.com/free-vector/network-mainframe-infrastructure-server-room-datacenter_39422-480.jpg" width="50"> Tech Stack
* Python
* Pandas
* Matplotlib

## How to run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python main.py
